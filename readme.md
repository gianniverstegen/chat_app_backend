# Backend for my global chatroom app

## Technologies used:

- TypeScript
- Node.js
- TypeORM
- GraphQL
- Apollo Server
- Redis
- Postgres
- Argon2

## Main functionality:

- Handle login and registration
- Store users and messages to DB
- Retrieve user info and messages from DB
- Keep track of user sessions
